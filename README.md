## 📌 Overview

Saarthi is an AI-powered digital banking assistant designed to bridge the gap between users and digital banking services. It combines conversational AI, personalized learning, fraud awareness, and a realistic banking simulator to build confidence before users perform real banking transactions.

Unlike traditional banking chatbots that only answer queries, Saarthi educates, guides, and trains users through an interactive learning experience.

---

# 🎯 Problem Statement

Many users hesitate to use digital banking due to:

- Fear of online fraud
- Lack of digital literacy
- Complex banking applications
- No safe environment to practice banking
- Limited personalized guidance

These challenges especially affect senior citizens, first-time users, and people from rural communities.

---

# 💡 Our Solution

Saarthi serves as an intelligent banking mentor that helps users learn and practice digital banking in a safe environment.

It provides:

- 🤖 AI-powered banking assistant
- 📚 Personalized learning roadmap
- 🛡 Fraud detection and awareness
- 🏦 Interactive banking simulator
- 📈 Progress tracking and recommendations

# 🧠 Multi-Agent AI Architecture

The application follows a modular AI architecture.

### 🔹 Intent Detection Agent

Identifies whether the user needs:

- Learning
- Banking support
- Fraud awareness
- Transaction guidance
- General conversation

---

### 🔹 Learning Planner Agent

Creates personalized learning paths based on:

- User level
- Previous lessons
- Current progress

---

### 🔹 Fraud Detection Agent

Analyzes conversations and educates users about banking scams.

---

### 🔹 Recommendation Agent

Suggests:

- Learning modules
- Banking simulations
- Security tips
- Next learning steps

---

### 🔹 Memory Agent

Maintains:

- Conversation summary
- Current lesson
- Current learning step
- Adoption score
- User history

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/MoulyaKS243/Saarthi.git
```

Navigate to the project

```bash
cd Saarthi/frontend
```

Install dependencies

```bash
npm install
```

Create `.env.local`

```env
MONGODB_URI=your_mongodb_connection_string
GROQ_API_KEY=your_groq_api_key
```

Run the project

```bash
npm run dev
```

Open

```
http://localhost:3000
```
